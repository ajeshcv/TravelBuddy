# Travel Buddy Matcher

A full-stack travel companion matching platform built using:

- Spring Boot
- Spring Security
- JWT
- PostgreSQL
- React
- WebSocket
- STOMP

## Features

- User Registration
- Login Authentication
- JWT Security
- Travel Plans
- Travel Matching Algorithm
- Match Requests
- Real-Time Chat
- Travel Buddy Discovery

## Tech Stack

Backend:
- Java
- Spring Boot
- PostgreSQL

Frontend:
- React
- Axios
- React Router

Real Time:
- WebSocket
- STOMP
